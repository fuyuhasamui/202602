# 202602

職能レベル判定:本人コメントフォーマット

【はじめに】
このファイルは、職能レベル判定における「自己申告入力」をサポートするためのものです。
利用は必須ではありませんが、自己申告者がポイントを抑えた記載ができ、判定者に分かりやすく伝えるものです。
是非活用してください。

【使い方】
以下の内容を追記し、レベル毎の「本人コメント」に貼り付けしてください。
・チェック項目（【】内の記述）に該当する内容を記載してください。。
　それ以外があればフリーフォーマット、および「→」に続けてコメントを追記してください。

詳細定義
・専門分野における第一人者として、部門の技術ビジョンや開発計画の立案を図るとともに、新技術などの開発・設計・新企画の提案および推進などができる
・自らの専門性をさらに高めるべく、常に最新の知識・技術を取り入れることができる。
・過去にとらわれず本質的な課題を自ら形成することができ、創造性・独創性を盛り込みながら、業務遂行への反映や必要な方策の検討を行うことができる。
----------
L4-2
詳細定義
・プロジェクトの将来的な成長・発展に向けて自らの専門知識やノウハウを組織に開示し、製品開発やサービス開発に貢献することができる。
・障害に直面しても、状況を打開しながらプロジェクトを遂行するとともに、仕事のプロセスと結果を検証し、別の機会に活かすことができる。
----------
L4-3
詳細定義
・専門領域の有識者として国内外の業界または学会等で「その分野の権威者」として認知され、複数のプロジェクトの策定に関わり、専門分野での価値創造を通じて戦略の実現に貢献できる。
・専門分野において新たな顧客価値の創造につながる中長期的な戦略を描き、多様な技術課題に対して業務遂行ができ、お客様の上位層に提案し、その実現に結びつけることができる。

(1)お客様の問題を新たな切り口で革新的に解決することができる。

レベル５
------------------------------
定義
社内外に通用する専門性を保有しており、事業領域全体を視野に入れた技術課題をまとめ上げることができる。また、市場全体から見ても先進的な製品の開拓や市場化をリードした経験と実績を有しており、世界で通用する技術者として認められる。
------------------------------
L5-1
詳細定義
・お客様の付加価値を創造し、市場競争力を決定づける業務設計および指摘することができる。また、担当分野の中長期的ロードマップを策定し、組織的に展開・推進している。
・実務経験や高い習熟度を活かして、組織における技術課題をまとめ上げることができ、課題解決に向けた提言をする事が出来る。
----------
L5-2
詳細定義
・社内外に通用する専門性・識見を核として、事業領域を視野に入れた技術課題をまとめ上げお客様事業における技術コンサルテーション業務を遂行することができる。
・事業を取り巻く環境変化も踏まえて、指導力を発揮し組織における課題を解決することができる。
----------
L5-3
詳細定義
・社内外に通用する複数の専門性に基づく広い視野・知識をを持つとともに担当の専門分野において、「世界でもトップレベルの技術力」を保有し、事業領域全体を視野に入れた技術課題をまとめ上げることができる。
・社会情勢・環境変化を踏まえて組織の枠組みを超えて指導力を発揮し、技術を通して経営的・社会的課題の解決に貢献することができる。
----------
(1)高度な専門知識と豊富な経験を有し、お客様事業における技術コンサルテーション業務を遂行することができる。
→
(2)保有する技術を発展させて、新しい技術を生み出すことができる。
→
(3)保有する専門性を核として、社内外の関連部門に広範囲に影響を与える行動を常時行っている。
